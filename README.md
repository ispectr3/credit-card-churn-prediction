# Credit Card Churn Prediction

> An end-to-end Machine Learning pipeline to predict credit card customer churn, extract business rules, and provide AI explainability using SHAP.

## Project Overview
Customer retention is crucial in the banking sector. This project leverages historical demographic and transactional data to predict which credit card customers are highly likely to cancel their services (Churn). By accurately identifying these at-risk customers, banks can proactively offer targeted incentives to retain them.

### Key Highlights
- **Direct Data Integration:** Dynamically pulls the official dataset directly from Kaggle via the `kagglehub` API.
- **Handling Imbalanced Data:** The dataset is heavily imbalanced (~16% churners). The model is designed to handle this discrepancy effectively.
- **Explainable AI (XAI):** We don't just predict *who* will churn; we explain *why*. By utilizing **SHAP Values**, the model's black box is broken down into clear, actionable business rules.

## Technologies & Tools
- **Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Random Forest)
- **Model Explainability:** SHAP
- **Visualization:** Matplotlib, Seaborn

## Dataset
The original dataset is hosted on Kaggle: [Credit Card Customers](https://www.kaggle.com/datasets/sakshigoyal7/credit-card-customers).
It contains 10,127 records and 21 features encompassing demographics (Age, Gender, Marital Status) and product usage (Credit Limit, Transaction Amount, Contact Frequency).
